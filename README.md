# india-urban-demographics-dashboard
Interactive Power BI dashboard analyzing population, literacy, education, and gender indicators across Indian cities.

## Project Overview
Interactive Power BI dashboard analyzing population, literacy, graduates, gender ratio, and child gender ratio across 492 Indian cities.

## Tools Used
- Power BI
- DAX
- Data Modeling
- Data Visualization

## Dashboard Features
- KPI Cards
- Interactive Map
- State & City Slicers
- Population Analysis
- Literacy Analysis
- Gender Ratio Analysis
- Treemap
- Scatter Plot

## Key Insights
- Maharashtra has the highest urban population.
- Literacy rates vary significantly across cities.
- Graduate population is concentrated in major metropolitan areas.
